Syntax

Create numbers and Booleans using numbers and the Boolean keywords.

You can declare numbers by just typing numbers (42 and 3.14159). Booleans are declared with the keywords true and false.

  1. In the console, type a number that is an integer.
  2. In the console, type a number that is a decimal.
  3. Type each of the Boolean flags in the console.
