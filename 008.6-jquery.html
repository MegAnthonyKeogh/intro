jQuery show()

Let's invoke a function on the jQuery show() animation!
We're completing these tasks in Submlime.

Show Image

This is an example of a click function:

    $("mySelector").show(300);

There is a hidden image above that has an ID of image.

  1. Open the js/008.6-jquery.js file in Sublime Text. It should have a ready() function and a click() function.
  2. Inside the click() function on the #button element, add the show() function above using the selector #image.
    • Hint: Add the new code right below the comment // add show event here.
  3. Save the js/008.6-jquery.js file and refresh this page in your browser.
  4. Click on the button above. The image should appear in 300 milliseconds.
