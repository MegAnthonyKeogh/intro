Syntax

Create strings in JavaScript with the reserved characters.

In JavaScript, you can declare strings with single or double quotation marks. A backward slash (\) can escape a character.

  1. Open the console in Chrome.
  2. Type a string in the console with double quotation marks.
    • Hint: "String goes between the double quotation marks."
  3. Type a string in the console with single quotation marks.
  4. Type another string in the console with single quotation marks. This time, add a single quote mark inside the string.
    • Hint: You need to escape the single quote mark inside the string.
