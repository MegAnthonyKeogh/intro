Let's add jQuery to our page!
We're now completing our work in Submlime and the console.

This function has been added to the page:

    <script src="https://ajax.googleapis.com/ajax/libs/jquery/2.1.3/jquery.min.js"></script>
  1. Open the 006-jquery-cdn.html file in Sublime Text. Add the script to the jQuery file from the Google CDN right before the closing body tag.
    • Hint: Look for the comment "ADD JQUERY CDN HERE"
  2. Save the 006-jquery-cdn.html file and refresh this page in your browser.
  3. In the console, execute $.fn.jquery. If this returns a version number, you have succesfully added jQuery to this page!.
